# House Price Prediction

## Overview
This project focuses on predicting house prices using machine learning techniques.

## Problem Statement
Predicting house prices helps in better decision making for buyers and sellers.

## Approach
- Data cleaning and preprocessing
- Feature engineering
- Applied regression models (Linear, Ridge, Lasso)
- Model evaluation

## Tools Used
- Python
- Pandas
- Scikit-learn

## Result
Built models to predict house prices with improved accuracy.
